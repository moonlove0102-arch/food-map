# 鳴川拉麵

## 餐廳資訊
- **類型**: 平價拉麵 / 自助吧
- **特色**: 拉麵 125 元起，內用雪花冰、飲料吃到飽！炸物炸得超讚，幸運兒還有機會再來一碗。
- **地址**: 新北市蘆洲區光華路161號
- **電話**: 02-8281-1393
- **營業時間**: 11:30 - 21:30
- **備註**: 內用限時 90 分鐘、無 10% 服務費
- **相關連結**: [Instagram Reel 介紹影片](https://www.instagram.com/reel/DYLJycmT5lq/?igsh=enBlMGQ4ZTUyaTZn)
