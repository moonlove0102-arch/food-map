# 肥前屋

## 餐廳資訊
- **類型**: 日式鰻魚飯 / 定食
- **特色**: 台北老字號鰻魚飯名店，停業一年後於 2026 年 3 月重新開幕，改善了過去被詬病的魚刺問題，重開後吸引大量人潮。好吃到讓人隔天還想帶朋友再去。
- **地址**: 台北市中山區中山北路一段121巷13號（條通商圈內）
- **營業時間**: 11:30 - 14:30、17:30 - 20:30（週二公休）
- **付款方式**: 僅收現金
- **訂位**: 不提供訂位，採現場排隊制
- **相關連結**: [Instagram Reel 介紹影片](https://www.instagram.com/reel/DX83V87zyJ6/?igsh=dW41N3IzMjlsNHJt)
